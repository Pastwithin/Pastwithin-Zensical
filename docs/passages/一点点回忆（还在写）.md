# 一点点回忆（还在写）



​	看到98上一篇回忆过去的文章，大概是很有感触吧，看完之后我去看了我们初中的校庆的视频。其实这个视频，大概出来有几天了，当时我没打开看。主要是，我其实还是不明白我该怎么看待我的初中。

​	从我初三的最后的选择说起吧，那个时候，对于平时成绩还不错的学生来说，他们算是有两个选择吧：一种，是大概在五月初就去到县里边的高中，算是一种提前招生吧；另一种是去到市里边的高中。不过，一直以来，县里边成绩靠前的学生，大都是选择出去到市里边去的。这算是一直以来的一种氛围吧，在初中的氛围的影响下，这好像就是一件很自然的事情。以至于，其实到我最后做决定之前的两个周之前，我其实都没有认真的考虑过，我会留在县中。

​	当时在几个老师们口中，留在县中好像就是不如去市里边。最后大家心里好像也就是那么想的。

​	那我最后选择留下，其实出乎了很多人的意料，我想说，你给刚刚进入初三的我说，我最后会留在县中，我也不会相信。好像在之前，成绩比较好的同学就是应该出去读书，在我们的心里好像就是那么理所应当的一样。但最后我为什么会选择留下呢？

​	其实我自己也不是完全能够说清楚，或许是当时我爸给我讲的那些出去读书但是最后也不是很成功的例子让我对自己出去读书之后能不能考好感到很没底；或许是我自己在去了市里边那两所高中参观了之后感到其实出去读也不是老师口中的那回事儿。总之就是突然之间，之前的那些想法一下子就全部被推倒了。我自己其实感受到，其实没有什么特别坚实的理由，来支撑我要出去到市里边的中学去读书。

​	所以说，在初三那年的劳动节假期之后，我就去了我之后的高中。

​	可听我们同学说，假期之后他们上课的有一天，班主任一天上课上到一半，突然情绪很激动，说了一些话，大概是说，我们几个去县中的同学不该走，他当时就不应该让我们去参加县中组织的选拔考试之类的话。乃至于之后其实还给我家长打过电话，想要劝我回去。我可能比较敏感吧，从这件事情里边，我感受到的其实是一种“被否定”。这一度让那段时间的我很迷茫，就算是这个决定是我自己做出来的，其实我也还是多多少少会受到一些过去的想法的影响，其实有的时候会有那么一点点动摇。但当我后来看开了之后，其实我感受到，在我们的初中，其实就是人为地，制造了一种对比，制造了一种焦虑，好像去县中就是会不如去市里边的中学好。这是一件没有明说，但是其实大家好像早就觉得很自然的东西。

​	虽然他们是我们初中的时候的引路人，但是难道他们就能够去定义我们高中的选择的优劣了？

​	有一些事情，我现在想起来，会觉得很荒唐很压抑，但是当时我居然没有察觉到？我记得有一次考得不太好，在这次考试之前，学校有一次艺术节，我和另外一个同学，找我们班主任，借了象棋去下。在这之后，班里不知道是谁带了一副象棋来，下午饭后，有时会有同学下棋玩。也就因此，我们班主任就认定，我考差，是因为象棋。但是其实，我是从年级第二考到第八；从艺术节之后，我也就下了半盘象棋。但是，我们班主任，在之后就经常，拿这件事情来，怎么说呢，我觉得是嘲讽我。我印象最深的，是我一次去上厕所，路上遇到他，他一句：“又去找人下象棋啦？”让我记到了现在。我也记得，当时班上有一个同学，大概是成绩算不上好，但是比较喜欢各种课外的知识。也是，有时会被在全班同学面前，我可以说，也是像之前那件事情一样的，没有必要的讥讽吧。作为老师，对那个同学有一种恨铁不成钢的感觉，我觉得是正常的，但是难道那个同学就应该在大家面前被那么去对待吗？从我作为一个旁观者的角度，我真的觉得，那是对那个同学的一种伤害啊。

​	还有一次是在一次初三的考试之后吧，校领导把成绩比较好的几个同学叫过去，说的事情，主要的意思是希望我们在之后能考好一些，上次我们学校在县里边的前十的排名情况不是很好。其实在这里我真的从里边看到了一点黑塞的《在轮下》的影子，书里边有一个人，作者这么说他：“如果他有得选，他宁愿学习一半的知识考全班第一，也不愿学双倍的知识但是屈居第二”。我感觉这不是很像当时我们的领导的心态吗？把这件事情和当时我去了县中之后，校领导曾经想过要把我劝回来联系起来，我很难不怀疑，其实他们当时的一部分考量是为了能够让学校中考的时候能够交出一份更好的成绩单，写在喜报上边的去市里边的中学就读的人数能够多一个。

​	还有好多当时班上的一些事情，就不细说了。

​	后边在我高三的时候，听说了一件其实让我很炸的事情，大概是，我初中的数学老师，对现在的他的学生说，我被我爸“耽误了”：因为我爸是公职人员，所以我必须要留在县中（但其实这是完全没有的事情）。

​	刚刚听到这件事情的时候我真的很生气。我觉得他不应该怎么说我爸，即使是有这么一件事，况且，那是完全就没有的事情。我们上一届的中考第二名，也是公职人员子女啊，也是出去读的啊，我觉得这件事情他不可能不知道。我也觉得很割裂，不知道该怎么看待，因为我和我们数学老师关系算比较好的了吧，遇到什么节日的时候，我还会想起来给他打个电话什么的。但是有了这件事情，我真不知道该说什么。到了后边，我想，其实可能他也知道那不是真的，只是为了给下一届的学生表达出他认为的去市中会更好的想法。

​	我还能再说什么呢？就算是每一年，统计出来的出去读书的学生最后的高考成绩对比下来其实都不是很理想；就算是我觉得我自己在县中取得了还不错的成绩，他们也还是认为去市里边的中学会更好。

​	我写这些，不是想去说我留在县中就是一个更好的选择，我觉得这最后其实还要分人来看的。我想要批评的是，我们的初中，好像总是只去强调成绩，好像就是认为去市里边的中学就是好学生唯一的选择，不这么选好像就是“糊涂”一样。

​	我知道这背后其实是有很多社会因素的：就好像对于高中，大家总是把更多的目光放在这个学校又出了多少个清华北大，对于我们当时的初中，大家也总是去看，这个初中又出了多少个到市里边读书的学生。好像其它到县中的同学，他们就不值得被关注一样。但是我觉得，社会上边的这种期待不应该那么影响到真正做出决策、真正要为自己的决策负责的学生。然而当时我们的初中是怎么做的呢？他们不管是不是合适，总都劝学生出去。我的上一届有一个学长，家里条件其实不太好，成绩也大概是刚刚能够过市中的线，从家里到市里边大概都要两三个小时的车程，最后是被老师劝着出去读——你可以想象他在他的整个高中，其实很难回一趟家，他在。在每年中考的喜报上边的多少人在市中就读的背后，三年后多少不适合的人的悲哀呢？我也觉得，他们不应该那样的只是强调成绩，好像你只需要成绩好；好像你所有的时间都应该放在学习上。我还觉得，他们不应该人为的营造出那样的氛围，让我们那些留在县中的学生就感到一种被否定。

​	回想这些事情，我总是会想到《在轮下》里的一些情节，倒是没有那么压抑。只是，我觉得里边很多的东西好像都是来自一样的东西，带给我一样的无力感。

​	这就让我对我初中的看法非常拧巴，一方面我觉得我的老师都是很负责的老师，他们中的很多人也确实对我有很大的影响。当我在看那个校庆的视频的时候，看到熟悉的的老师的时候，那个时候其实我一度觉得我真的放下了之前的那些事情。但是当我再想的时候，我又觉得，还是很难去接受。我的意思是，当我想要去接纳那段过去的时候，想起来的这些一件件事情，却让我感到，好像我在那个时候就没有被他们接纳过。

​	就先写到这里吧。
