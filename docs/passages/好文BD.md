# 回忆种种总是慨叹又微笑

作者：飘渺启真湖<br>来源：cc98

​	熬夜，熬夜，再熬夜，成了这段时间的主旋律，随之而来的是体重的上升和外表的疲态，凌晨冲回寝室的时候，整个人已经是一种恍惚的状态了，然而也正因为恍惚，产生了对周围世界的剥离感和时间的距离感，就好比做梦刚醒来的时候，自己的身份和现实中的各种窘境逐渐袭来，不由得回忆起从小到大的种种，时间流逝得太快，导致之前的记忆慢慢褪色了，我很早就意识到，我们的人生从某个时间开始，厚度将不再增加，当新的记忆产生的速率无法抵消之前记忆衰减的速率时，我们本质上就是穿行在悠长的时间隧道，黑漆漆一片只找得亮前面和后面的路，而且我们的灯还是逐渐熄灭的。我现在回忆时候总是带着恐惧，怕记忆像磁带会慢慢磨损所以情愿不去读它。 

​	小时候我们体验到“初次”，老了会体验到“末次”，我们对“出生”没有把握的机会，但是对死亡却又可能意识清醒地感受到，希望如此吧！我还记得我的一位长辈，我姥姥的妈妈，在九十多岁的高龄去世，死前还很清醒，我妈妈后来说，我的太姥姥（这个叫法太生硬了）死前认为自己还能存活，并且因此对她的孩子颇为怨恨，虽然她已经许多天不吃饭了。她是个享福的人，一辈子没有干过活，没有做过饭，我未曾谋面的太姥爷包揽一切。记得小时候我给她送了点吃的，她笑眯眯端详我，说着“咱娃真兴人......长得真高"突然问”你是谁的孩子？“她不是糊涂，而是辨认重孙辈确有困难，我妈妈至今说起来这件事都会笑。 

​	也有莫名其妙就去世的时候吧，我的舅舅就是这样去世的，之前谁都没有想到，我姥姥是最后一个知道这件事的，这之前她二儿子旁敲侧击地说过，说村里的某某，儿子年轻轻的就走了，我姥姥接话了，”世界上哪里还能有比这更闹心的事情呢？“其实，知道人死了，对活着的人来说是他/她死的第一步，见到遗体，是死的第二步，火化或者埋葬，是死的结束。我家人去舅舅工作的地方把他的遗体带回来，我听说，他们让他放在后排，靠在他的兄弟们的身边，他哥哥说，三小子回家了。后来他们给我舅舅配了冥婚，我这位舅妈没有见过夫君，我舅舅也没见过娘子，我初听到这个消息感觉有点诡异，后来又想也好，他俩在黄泉路上走着走着就熟悉了，不然一个人多么孤单啊。 

​	知道我舅舅去世，我妈妈以泪洗面，后来她去逛商场，买了件新衣服，我说她看起来好多了，她脸上划过一丝愧疚——我现在知道，那是对无法永远留在悲痛情绪的愧疚，我深刻理解这种无力，如果人生只有一天多好，感动和悲喜将不再担心遗忘，我们不用一个又一个人为划定的纪念日和节日来刺破过长时间带来的麻木。 

​	宴席散去的尴尬，熬了大夜将天亮的后悔，还有情人的变心，还有昔日志士的蜕变，假如时间不这么长多好啊，永远留在热恋的美好，永远留在青春的激情，那样，汪精卫就成了英雄，李隆基也成了完人。可惜进度条拉过去，艳电发送了，长安陷落了，爱人老去了，连接吻都会累。有的历史看了会叹息，他/她怎么不多活几年，有的历史看了会痛批，他/她怎么不那个时候就死（谁把“他”分给了男人？害得我必须这样打），我想梁武帝饿死前也一定会闪过类似的念头吧，八十多岁的人那么不体面，我看到曹丕那么早过世，也忍不住想，永远留在“陛下临轩笑”，多好。 

​	但是其实我妈妈不会遗忘去世的弟弟，她现在想起来也依然会叹息，我姥姥更不用说，焚心之苦莫过于此。记得同村有个乡亲，没有儿子，她很羡慕我姥姥，常说，弟兄三个回来，一个个高高壮壮的站在堂前，哪里有更幸福的事情，多么幸福的事情，后来她也安慰说，你还有两个呢，我孤零零的谁都没有。可是谁都代替不了另一个人啊，越是年龄大的，就越受不了离别，更接受不了生离死别，因为身边人，是行走在时间隧道的锚点，你忘了之前的事情，问问锚点就什么都知道了，某个锚点消失了，某些记忆就找不回了，生命的厚度就丢失了。 

​	写到这里我又想起我舅舅的事，他和我睡在一起的时候，我用他的手机打开了王者荣耀，四五分都没有走出泉水，我不懂这是什么游戏，难道就是走来走去吗，后面被他叫一声拿走了，还有他之前用黄瓜蘸黄豆酱，还有他曾经直接对着宁化府的瓶子吹醋......他当时站在堂前红柜子那里喝的，那个柜子底下曾经有我抓的一只小麻雀宝宝，它怕我，躲在了底下，后来被猫一爪子抓出吃掉了，我现在还很愧疚，我送走了小麻雀的一生。

​	 如果死了有鬼就好了，真的求求了，一定要有吧，如果有鬼，起码能证明，我们的存在不是一个缓慢氧化的过程，我们的智慧悲欢也不是巨分子反应的偶然，求你了，让鬼存在吧，我以后变个小鬼也可以。我一个亲戚说她梦见过去世的丈夫，而且她很确定，他托梦来了。我妈妈很嗔怪不让我听这种事情。我爸爸对祖先，家族这类看得很重，不知道他信不信有鬼？我妈妈讲过，她之前骑着电动车，路上遇到一个骑摩托车的背影，很像她爸爸，她一边哭一边奋力追赶，那时候她真相信了，那就是他，连摩托车款式都一样，她哭着哭着，追上去，结果不是他，又哭着走过了。 

​	现在写下还是很难受，为这许多爱恨悲欢，虽然我还没走过隧道的多少路，但积压在记忆里的感叹是能遗传的，这场阴谋藏在耳听口传，藏在卷卷历史，不经意瞟过一眼就成了之后某个凌晨的难过，刺破心防。无数颗心脏在跳动，把血往上泵入大脑，最后我们自食苦果，损神熬心，偏偏乐在其中。
