常微分方程

§1.基本概念

1.1 两个实例

例 1. 冷却问题

  • 设有温度为 100℃ 的物体放置在 20℃ 的空气中冷却，求物体温度 \(T\) 与时间 \(t\) 的关系。（假设空气恒温，且根据冷却定理，物体温度的变化率和物体与介质间的温度差成正比）

解：设比例系数为 k(k > 0) \(\begin{cases} \frac{\mathrm{d} T}{\mathrm{d} t}=-k(T-20) （1） \\T\mid_{t = 0} = 100 （2） \end{cases}\)​​

由（1）式：

$\large \frac{\mathrm{d} T}{T-20} = -k\mathrm{{d} t}\longrightarrow \int \frac{\mathrm{d} T}{T-20}=\int -k\mathrm{{d} t} $

\(\longrightarrow ln(T-20)=-kt+C_{1}\longrightarrow T-20 = e^{C_{1}}e^{-kt}\) ​

再代入（2）式，可以求出 C = 100-20 = 80.故：\(T=80e^{-kt}+20\)

例 2.自由落体问题

  • 设有质量为 \(m\) 的物体只受重力作用由静止自由降落，求其降落距离和时间的关系。

解：设时刻 t 下物体降落的距离为 x.

\(\begin{cases}\large \frac{\mathrm{d^{2}} x}{\mathrm{d} t^{2}}= g \\\large x\mid_{t = 0}= 0 \\\large \frac{\mathrm{d} x}{\mathrm{d} t}\mid_{t = 0}= 0 \end{cases}\)

由第一个式子：\(\large \frac{\mathrm{d} x}{\mathrm {d} t}= gt+C_{1} \longrightarrow x =\frac{1}{2}gt^{2}+C_{1}t+C_{2}\)

再由两个初值条件可以解出 \(C_{1}=C_{2}=0\).

得：\(x=\frac{1}{2}gt^{2}\)

1.2 基本概念

1.微分方程定义

联系着 自变量、因变量及因变量的导数 的方程叫做微分方程。 - 微分方程的形式：\(F(x,y,y',y''…y^{n})=0\). - 常微分方程：未知函数是一元函数（自变量只有一个）。 - 偏微分方程：未知函数是多元函数（自变量有很多个）。

2.微分方程的阶数

微分方程的阶数等于因变量的 最高阶导数的阶数。

​ 例：\(\large\frac{\mathrm{d^{2} y}}{\mathrm{d} x^{2}}=\frac{1}{a}\sqrt{1+(\frac{\mathrm{d} y}{\mathrm{d} x})^{3}}\) 是二阶微分方程。

3.n 阶微分方程的通解

若 n 阶微分方程的解中含有 n 个 相互独立 的任意常数，则称其为这个 n 阶微分方程的通解。

可记作：\(\varphi(x,y,C_{1},C_{2}…,C_{n})=0\)

4.微分方程的特解

不含任意常数的解

5.定解条件

​ 通过其它附加条件可以求出特解，这些条件称为定解条件。

  • 初值条件： 定解条件都 在一个初始点 上，例如： $$ \begin{cases} y'= 2x \y(0)= 1 \end{cases} $$ 这样的问题称为初值问题

  • 边值问题： 定解条件 在不同的点 上，例如： $$ \begin{cases} y''= p(x)y'+q(x)y = r(x) \y(a)=\alpha, y(b)=\beta \end{cases} $$ 这样的问题叫做边值问题

[!note]

书上的定理 1.1 和定理 1.2 实际上描述了：在一定条件下，初值问题存在唯一的解。

§2.可分离变量方程、变量型齐次方程

2.1 可分离变量方程

1.形式

​ \(\large\frac{\mathrm{d} y}{\mathrm{d} x}=f(x)g(y)\)

  • 特点:
  • 已经解出一阶导数
  • 右端是 \(x\) 的一元函数与 \(y\) 的一元函数的乘积的形式

2.解法

分离变量法

（1）\(g(y) \ne 0\)

​ \(\large\frac{\mathrm{d} y}{\mathrm{d} x}=f(x)g(y)\longrightarrow\frac{\mathrm{d} y}{g(y)}=f(x)\mathrm{d}x\longrightarrow\int\frac{\mathrm{d} y}{g(y)}=\int f(x)\mathrm{d}x+C\)

化简后即可得到通解.

(2) 若当 \(y=y_{0}\) 时，有 \(g(y_{0})=0\) 时，则 \(y=y_{0}\) 也是方程的一个通解

[!note]

这个解可能不包含于（1）中通解，故需单独验证。

​ 流程： 判断方程类型 \(\longrightarrow\) 分离变量 \(\longrightarrow\) 两边求积分 \(\longrightarrow\) 化简得到通解 \(\longrightarrow\) 若有初值条件，求出特解即可；若无初值条件，则需要考虑 \(g(y)=0\) 的根，当 \(g(y_{0})=0\) 则 \(y=y_{0}\) 也是方程的解。

3.例子

例 1 求 \(\large\tan{x}\frac{\mathrm{d} y}{\mathrm{d} x}=1+y\) 通解

解：\(\large\tan{x}\frac{\mathrm{d} y}{\mathrm{d} x}=1+y\longrightarrow\large\frac{\mathrm{d} y}{1+y}=\frac{cos{x}}{sin{x}}\mathrm{d}x\longrightarrow\int \frac{\mathrm{d} y}{1+y}=\int \frac{cos{x}}{sin{x}}\mathrm{d}x\longrightarrow ln|1+y|=ln|sinx|+C_{1}\longrightarrow 1+y=\pm e^{C_{1}}sinx\)

令 \(C=\pm e^{C_{1}}(C\ne 0)\)

可得 \(\large1+y=csinx\longrightarrow y=C sinx-1(C\ne 0)\)

又：\(y=-1\) 也是方程的解，此时对应上式中 \(C=0\)

故通解为 \(y=Csinx-1\)

[!caution]

在常微分方程中，\(\large \int \frac{\mathrm{d}x}{g(x)}\) 代表的不再是全体原函数，而是一个任意但是确定的原函数。

[!note]

在两边求积分之后，加积分常数的时候，可以根据等式的形式灵活处理所加常数 \(C\) 的形式

​ 例如: 上式中如果将所加常数 \(C_{1}\) 从一开始就替换成 \(ln|C_{1}|\) \((C_{1}\ne0)\) 就会简化很多，同理，对于 \(-ln|1+y|=ln|sinx|+C\)，替换所加常数为 \(-ln|C|\)，即写成 \(-ln|1+y|=ln|sinx|-ln|C|\)，对于 \(\frac{1}{2}ln|1+y|=ln|sinx|+C\)，将所加常数替换为 \(\frac{1}{2}ln|C|\), 即写成 \(\frac{1}{2}ln|1+y|=ln|sinx|+\frac{1}{2}ln|C|\), 都会是更好的选择。

例 2 求 \(\large\mathrm{d} x+xy\mathrm{d}y=y^{2}\mathrm{d}x+y\mathrm{d}y\) 通解

解：\(\large\mathrm{d} x+xy\mathrm{d}y=y^{2}\mathrm{d}x+y\mathrm{d}y\longrightarrow \frac{\mathrm{d} y}{\mathrm{d} x}=\frac{1}{x-1}(y-\frac{1}{y})\longrightarrow \frac{y\mathrm{d} y}{y^{2}-1}=\frac{\mathrm{d} x}{x-1}\longrightarrow\)

\(\int \frac{y\mathrm{d} y}{y^{2}-1}= \int \frac{\mathrm{d} x}{x-1}\longrightarrow ln|y^2-1|=ln(x-1)^2+ln|C|\longrightarrow y^{2}-1=C(x-1)^2\) \((C\ne 0)\)

(此处不写成 \(\pm C(x-1)^2\) 是由于 \(C\) 是任意的常数，中间其实省略了一步)

再考虑 \(g(y)\) 零点，\(y=\pm1\) 也是方程通解，对应上式中 \(C=0\) 的情况

故通解为 \(y^{2}-1=C(x-1)^2\)

2.2 齐次方程

1.形式：

\(\Large \frac{\mathrm{d} y}{\mathrm{d} x} = g(\frac{y} {x})\)

2.解法：

变量替换法

​ 令 \(\large u=\frac{y}{x}\) , 则 \(y=ux\), \(\large \frac{\mathrm{d} y}{\mathrm {d} x}=\frac{\mathrm{d} ux}{\mathrm {d}x}=u+x\frac{\mathrm{d} u}{\mathrm{d}x}=g(u)\longrightarrow \frac{\mathrm{d} u}{\mathrm{d} x}=\frac{1}{x}(g(u)-u)\)​

可以看到变换成了一个可分离变量方程，按照可分离变量方程方式求解，最后再进行变量回带即可。

3.例子

例 3 求 \(\Large \frac{\mathrm{d} y}{\mathrm{ d} x}=\frac{y}{x}+tan\frac{y}{x}\) 且 \(y\mid_{x=1}=6\) 的特解.

解：令 \(\Large u=\frac{y}{x}\) , 则 \(y=ux\).

\(\large x \frac{\mathrm{d} u}{\mathrm{d} x} + u =u+tanu\longrightarrow \frac{\mathrm{d} u}{tanu}=\frac{\mathrm{d} x}{x}\longrightarrow \int \frac{\mathrm{d} u}{tanu}= \int \frac{\mathrm{d} x}{x}\longrightarrow ln|sinu|=ln|x|+ln|C|\longrightarrow sinu=Cx\) \((C\ne 0)\)

即 \(\large sin{\frac{y}{x}}=Cx\), 由初值条件：\(y\mid_{x=1}=6\), 带入求得 \(\large sin{\frac{\pi}{6}}=C·1=\frac{1}{2}\longrightarrow C=\frac{1}{2}\)

从而得到特解为 \(\large sin{\frac{y}{x}}=\frac{1}{2}x\)

[!NOTE]

(1).一定要记得变量回带

(2).在有初值条件的时候不用再讨论常数解的情况

(3).在一些特殊情况下令 \(\Large v=\frac{x}{y}\) 会简化运算，例如：\(\Large \frac{\mathrm{d}y}{\mathrm {d}x}=\frac{y}{x-\sqrt{x^2+y^2}}\) 如果令 \(\Large u=\frac{y}{x}\) 计算量会比较大，若将式子改写成 \(\Large \frac{\mathrm{d} x}{\mathrm{d} y}=\frac{x-\sqrt{x^2+y^2}}{y}\)，并且令 \(\Large v = \frac{x}{y}\)，计算会简便很多。 (保证分母形式简洁)

§3.一阶线性方程

3.1 一阶线性方程

1.形式

​ \(\large \frac{\mathrm{d} y}{\mathrm {d} x}+p(x)·y=q(x)\)

(1).当 \(q(x)\equiv0\) 时，称其为一阶齐次线性微分方程； (2).当 \(q(x)\) 不恒为 \(0\) 时，称其为一阶齐次线性微分方程。

2.解法

常数变易法

(1).先解齐次线性微分方程 \(\large \frac{\mathrm{d} y}{\mathrm{d} x}+p(x)·y=0\)

\(\large \frac{\mathrm{d} y}{\mathrm{d} x}+p(x)·y=0\longrightarrow \frac{\mathrm{d} y}{y}=-p(x)\mathrm{d} x\longrightarrow \int \frac{\mathrm{d} y}{y}=-\int p(x)\mathrm{d} x\longrightarrow ln|y|=-\int p(x)\mathrm{d} x+ln|C|\longrightarrow y=Ce^{-\int p(x)dx}\)

可得通解为 \(y=Ce^{-\int p(x)dx}\)

(2).设 \(y=ue^{-\int p(x)dx}\) (\(u\) 是一个 \(x\) 的一元函数), 是非齐次线性微分方程的解 (个人理解，这个解法类似于待定系数法)

带入原方程：\(\large \frac{\mathrm {d} u}{\mathrm {d} x}e^{-\int p(x)\mathrm {d} x}+ue^{-\int p(x)\mathrm{d} x}(-p(x))+p(x)·ue^{-\int p(x)\mathrm{d} x}=q(x)\longrightarrow \frac{\mathrm {d} u}{\mathrm {d} x}=q(x)e^{\int p(x)\mathrm {d} x}\longrightarrow u=\int q(x)e^{\int p(x)\mathrm{d}x}\mathrm{d}x+C\)

则 \(y=e^{-\int p(x)\mathrm{d} x}[\int q(x)·e^{\int p(x)\mathrm{d}x}\mathrm {d}x+C]\) 为通解.

[!note]

在假设完通解之后再带入原方程之后，可以通过检查是否相抵来检查自己的计算。

流程： 第一步先求出对应的齐次线性微分方程的通解，第二步再使用常数变易法。

3.一阶线性非齐次微分方程的通解的结构（公式）

对于 \(\Large \frac{\mathrm{d} y}{\mathrm {d} x}+p(x)·y=q(x)\)

\(y=e^{-\int p(x)\mathrm{d} x}[\int q(x)·e^{\int p(x)\mathrm{d}x}\mathrm {d}x+C]=e^{-\int p(x)\mathrm{d} x}\int q(x)·e^{\int p(x)\mathrm{d}x}\mathrm {d}x+Ce^{-\int p(x)\mathrm{d} x}\) 是其通解.

可以注意到，左边的项实际上是一阶线性非齐次微分方程组的 特解，右边的项是一阶线性齐次方程组的 通解.

学过线性代数的同学应该会觉得这很眼熟。

以上的式子也给出了一阶线性微分方程的另一种解法：公式法

4.例子

例 1 求 \(\Large \frac{\mathrm {d} y}{\mathrm {d} x}+\frac{1}{x}·y=\large 5\) 的通解

解：先解 \(\Large \frac{\mathrm{d} y}{\mathrm {d} x}+\frac{1}{x}\large y=0\)​

\(\large \frac{\mathrm{d} y}{y}=-\frac{\mathrm{d}x}{x}\longrightarrow \int \frac{\mathrm{d} y}{y}=-\int \frac{\mathrm{d}x}{x}\longrightarrow ln|y|=-ln|x|+ln|C|\longrightarrow y=\frac{C}{x}\) \((C\ne 0)\)

当 \(y=0\) 时，对应上式中 \(C=0\) 的情形

即对应的齐次线性微分方程的通解为 \(\Large y=\frac{C}{x}\)

设原方程的通解为 \(\Large y=u·\frac{1}{x}\), 带入原方程，有：

\(\large \frac{\mathrm{d} u}{\mathrm{d} x}·\frac{1}{x}-u·\frac{1}{x^2}+u·\frac{1}{x^2}=5\longrightarrow \large \frac{\mathrm{d} u}{\mathrm{d} x}=5\longrightarrow \mathrm{d}u=5x\mathrm {d}x\longrightarrow \int \mathrm{d}u=\int 5x\mathrm {d}x\longrightarrow u=\frac{5}{2}x^{2}+C\)

所以通解为 \(y=\Large \frac{1}{x}(\frac{5}{2}x^2+\large C)\)

例 2 求 \(\Large 2x\frac{\mathrm{d} y}{\mathrm{d} x}-y=-x\) 的通解

解：原方程可以化为 \(\large \frac{\mathrm{d}y}{\mathrm{d}x}-\frac{1}{2x}y=-\frac{1}{2}\)

由上边的公式可得：

通解为: $ \large y=e^{\int \frac{1}{2x}\mathrm{d}x}[\int (-\frac{1}{2})e^{\int \frac{1}{2x}\mathrm{d}x}+\large C]\longrightarrow e^{\frac{1}{2}ln|x|}(-\frac{1}{2}\int e^{-\frac{1}{2}ln|x|}\mathrm {d}x+C)$

分类讨论 \(\Large x\) 的符号，去掉绝对值可得通解为 \(\large y=-x+C\sqrt{|x|}\) (\(x=0\) 的情况单独讨论)

3.2 伯努利方程

1.形式

​ \(\Large \frac{\mathrm{d}y}{\mathrm{d}x}+p(x)y=q(x)y^{\alpha}\) \((\alpha\ne 0,1)\)

[!note]

$\alpha =0 $ 时，方程即为一阶线性微分方程

\(\alpha=1\) 时，方程即为可分离变量微分方程

2.解法

变量替换法

\(\large \frac{\mathrm{d}y}{\mathrm{d}x}+p(x)y=q(x)y^{\alpha}\longrightarrow y^{-\alpha}\frac{\mathrm{d}y}{\mathrm{d}x}+p(x)y^{1-\alpha}=q(x)\)

凑微分：\(\Large \frac{1}{1-\alpha}·\frac{\mathrm{d} y^{1-\alpha}}{\mathrm{d}x}+p(x)y^{-\alpha}=q(x)\)

令 \(\Large z=y^{1-\alpha }\), 于是方程变为: \(\large \frac{\mathrm{d}z}{\mathrm{d}x}+(1-\alpha)p(x)·z=(1-\alpha)q(x)\)

即一个 \(z\) 关于 \(x\) 的一阶非齐次线性微分方程组，按照先前的解法，最后变量回代即可。

3.例子

例 3 求 \(\Large \frac{\mathrm{d}y}{\mathrm{d}x}+\frac{y}{x}=a(lnx)y^2\)

解: \(\large y^{-2}·\frac{\mathrm{d}y}{\mathrm{d}x}+y^{-1}·\frac{1}{x}=alnx\longrightarrow \frac{\mathrm {d}y^{-1}}{\mathrm{d}x}-\frac{y^{-1}}{x}=alnx\)

令 \(z=y^{-1}\)，则：\(\Large \frac{\mathrm {d} z}{\mathrm {d} x}-\frac{z}{x}=alnx\)

公式法: \(\large z=e^{\int \frac{1}{x}\mathrm{d}x}[\int (-alnx)·e^{\int -\frac{1}{x}\mathrm{d}x}+\large C]\longrightarrow \large z = x(-a\int \frac{1}{x}·lnx\mathrm{d}x+\large C)\longrightarrow \large z=x[-a·\frac{1}{2}(lnx)^2+\large C]\)

通解为 \(\large y=\frac{1}{x[-a·\frac{1}{2}(lnx)^2+\large C]}\)，此外 \(y=0\) 也是一个解.

[!note]

分母不为零的限定条件会引出一些需要单独讨论的解

§4.可降阶的二阶微分方程

4.1 \(\frac{\mathrm{d^2}y}{\mathrm{d}x^2}=f(x)\) 型

解法

连续积分法

\(\large \frac{\mathrm{d^2}y}{\mathrm{d}x^2}=f(x)\longrightarrow \frac{\mathrm{d}y}{\mathrm{d}x}=\int f(x)\mathrm{d}x+C_1\longrightarrow y=\int [\int f(x)\mathrm{d}x]\mathrm{d}x+C_{1}x+C_2\)​

例 1

求 \(\large \frac{\mathrm{d^2}y}{\mathrm{d}x^2}=\frac{1}{cos^2x}\) 且 \(y|_{x=\frac{\pi}{4}}=\frac{ln2}{2},\frac{\mathrm{d}y}{\mathrm{d}x}|_{x=\frac{\pi}{4}}=1\) 的特解.

解：\(\large \frac{\mathrm{d^2}y}{\mathrm{d}x^2}=\frac{1}{cos^2x}\longrightarrow \frac{\mathrm{d}y}{\mathrm{d}x}=tanx+C_1\)

由 \(\frac{\mathrm{d}y}{\mathrm{d}x}|_{x=\frac{\pi}{4}}=1\) 得 \(C_1=0\)

\(\frac{\mathrm{d}y}{\mathrm{d}x}=tanx\longrightarrow y=-ln|cosx|+C_2\)

由 \(y|_{x=\frac{\pi}{4}}=\frac{ln2}{2}\) ，得 \(C_2=0\)

从而得到特解为 \(y=-ln|cosx|\)

[!note]

使用类似的方法也可以解 \(\large \frac{\mathrm{d^n}y}{\mathrm {d}x^n}=f(x)\) 型

4.2 \(\frac{\mathrm{d^2}y}{\mathrm{d}x^2}=f(x,\frac{\mathrm{d}y}{\mathrm{d}x})\)​​ 型

特点： 不显含 \(y\)

解法

变量替换法

令：\(\large \frac{\mathrm{d}y}{\mathrm{d}x}=p\) , 则 \(\large \frac{\mathrm{d^2}y}{\mathrm{d}x^2}=\frac{\mathrm{d}p}{\mathrm{d}x}\)

方程可以化为：\(\large\frac{\mathrm{d}p}{\mathrm{d}x}=f(x,p)\) \((*)\)

设 \((*)\) 式的解为 \(p=\Phi(x,C_1)\), 即：\(\large\frac{\mathrm{d}y}{\mathrm{d}x}=\Phi(x,C_1)\)，即：\(\large y=\int \Phi(x,C_1)\mathrm{d}x+x+C_2\)

例 2

求 \(\large (1+x^2)\frac{\mathrm{d^2}y}{\mathrm{d}x^2}=2x\frac{\mathrm{d}y}{\mathrm{d}x}\) 的通解.

令：\(\large \frac{\mathrm{d}y}{\mathrm{d}x}=p\) , 则 \(\large \frac{\mathrm{d^2}y}{\mathrm{d}x^2}=\frac{\mathrm{d}p}{\mathrm{d}x}\)​

\(\large (1+x^2)\frac{\mathrm{d^2}y}{\mathrm{d}x^2}=2x\frac{\mathrm{d}y}{\mathrm{d}x}\longrightarrow (1+x^2)\frac{\mathrm{d}p}{\mathrm{d}x}=2xp\longrightarrow \frac{\mathrm{d}p}{p}=\frac{2x\mathrm{d}x}{1+x^2}\longrightarrow ln|p|=ln|1+x^2|+ln|C_1|\longrightarrow p=C_1(1+x^2)\)

(这里还应该单独讨论 \(y=0\) 的特殊情况)

即: \(\large \frac{\mathrm{d}y}{\mathrm{d}x}=C_1(1+x^2)\longrightarrow y=C_1(x+\frac{1}{3}x^3)+C_2\)

4.3 $ \frac{\mathrm {d^2}y}{\mathrm{d}x^2} = f(y,\frac{\mathrm{d}y}{\mathrm{d}x})$ 型

特点： 不显含 \(x\)

解法

变量替代法

令：\(\large \frac{\mathrm{d}y}{\mathrm{d}x}=p\) , 则 \(\large \frac{\mathrm{d^2}y}{\mathrm{d}x^2}=\frac{\mathrm{d}p}{\mathrm{d}x}=\frac{\mathrm{d}p}{\mathrm{d}y}·\frac{\mathrm{d}y}{\mathrm{d}x}=p·\frac{\mathrm{d}p}{\mathrm{d}x}\) （或者视为是复合函数求导过程）

方程可以化为：\(p·\large\frac{\mathrm{d}p}{\mathrm{d}x}=f(x,p)\) \((*)\)

设 \((*)\) 式的解为 \(p=\Phi(y,C_1)\), 则：\(\large\frac{\mathrm{d}y}{\mathrm{d}x}=\Phi(y,C_1)\longrightarrow \int\frac{\mathrm{d}y}{\Phi(y,C_1)}=\int \mathrm{d}x+C_2\)

例 3

求 \(\large (\frac{\mathrm{d}y}{\mathrm{d}x})^2-y·\frac{\mathrm{d^2}y}{\mathrm{d}x^2}=0\) 的通解

解：令：\(\large \frac{\mathrm{d}y}{\mathrm{d}x}=p\) , 则 \(\large \frac{\mathrm{d^2}y}{\mathrm{d}x^2}=\frac{\mathrm{d}p}{\mathrm{d}x}=\frac{\mathrm{d}p}{\mathrm{d}y}·\frac{\mathrm{d}y}{\mathrm{d}x}=p·\frac{\mathrm{d}p}{\mathrm{d}x}\)

原式化为：\(\large p^2-y·p·\frac{\mathrm{d}p}{\mathrm{d}y}=0\longrightarrow \frac{\mathrm{d}p}{p}=\frac{\mathrm{d}y}{y}\longrightarrow ln|p|=ln|y|+ln|C_1|\longrightarrow p=C_1y\)

(这里也应该单独讨论 \(y=0\) 的特殊情况)

即 \(\large \frac{\mathrm{d}y}{\mathrm{d}x}=C_1y\longrightarrow \frac{\mathrm{d}y}{y}=C_1\mathrm{d}x\longrightarrow ln|y|=C_1x+ln|C_2|\longrightarrow y=C_2·e^{C_1x}\) \((C_2\ne0)\)

又：\(y=0\) 的情况对应于上式中 \(C_2=0\) 的情况，故通解为 \(y=C_2·e^{C_1x}\)

§5.二阶线性微分方程的解的结构

5.1 二阶线性微分方程的形式

​ \(\large \frac{\mathrm{d^2}y}{\mathrm{d}x^2}+p(x)·\frac{\mathrm{d}y}{\mathrm{d}x}+q(x)·y=f(x)\)

(1).当 \(f(x)\equiv 0\), 则为二阶齐次线性微分方程。 (2).当 \(f(x)\) 不恒为 \(0\), 则为二阶非齐次线性微分方程。

5.2 二阶线性微分算子

符号： \(\large L=\frac{\mathrm{d^2}}{\mathrm{d}x^2}+p(x)·\frac{\mathrm{d}}{\mathrm{d}x}+q(x)\) 即：\(\large L[y]=\frac{\mathrm{d^2}y}{\mathrm{d}x^2}+p(x)·\frac{\mathrm{d}y}{\mathrm{d}x}+q(x)·y\)

于是上述式子可以简记为: \(\large L[y]=f(x)\)

性质： 讨论 \(L\) 的线性性质：

​ 设 \(y=y(x)\) \(y_1=y_1(x)\) \(y_2=y_2(x)\) 二阶可导，\(C,C_1,C_2\) 为常数，则有以下性质:

  • \(L[Cy]=CL[y].\)
  • \(L[y_1+y_2]=L[y_1]+L[y_2].\)
  • \(L[C_1y_1+C_2y_2]=C_1L[y_1]+C_2L[y_2].\)

5.3 叠加定理

定理 1： 设 \(y_1=y_1(x) , y_2=y_2(x)\) 都是 \(L[y]=0\) 的解，\(C_1,C_2\) 为常数，则 \(C_1y_1+C_2y_2\) 也是方程的解.

5.4 函数的线性相关与线性无关

​ 设 \(y_1=y_1(x) , y_2=y_2(x)\) 是定义在某区间 \(I\) 上的函数，若存在不全为 \(0\) 的常数 \(k_1,k_2\) , 使得 \(k_1y_1+k_2y_2=0\) , 则称 \(y_1(x)\) 与 \(y_2(x)\) 在区间 \(I\) 上线性相关，否则则称线性无关。

[!note]

​ 当 \(y_1(x),y_2(x)\) 至少有一个不为零时，可以通过判断两函数相除之后是否为常数的方式来判断是否线性相关

5.5 二阶齐次线性微分方程的通解的结构定理

定理 2： 设 \(y_1=y_1(x) , y_2=y_2(x)\) 是 \(L[y]=0\) 的两个线性无关的解，\(C_1,C_2\) 为任意常数，则 \(L[y]=0\) 的通解为 \(y=C_1y_1+C_2y_2\) .

5.6 二阶非齐次线性微分方程的通解的结构定理

定理 3： 设 \(\large \tilde{y}=\tilde y(x)\) 是 \(L[y]=f(x)\) 的一个特解，\(Y = C_1y_1+C_2y_2\) 是 \(L[y]=0\) 的通解，则：\(y=Y+\tilde {y}(x)=C_1y_1+C_2y_2+\tilde y(x)\) 为 \(L[y]=f(x)\) 的通解. 其中 \(C_1,C_2\) 为任意常数 .

5.7 广义叠加定理

定理 4： 设 \(y_1=y_1(x) , y_2=y_2(x)\) 分别是 \(L[y]=f(x),L[y]=g(x)\) 的解，则 \(y_1+y_2\) 是 \(L[y]=f(x)+g(x)\) 的解 .

定理 5： 设 \(y_1+iy_2\) 是方程 \(L[y]=f_1(x)+if_2(x)\) 的解，则 \(y_1,y_2\) 分别是 \(L[y]=f_1(x)\) 与 \(L[y]=f_2(x)\) 的解 .

§6. 二阶常系数微分方程

6.1 二阶常系数齐次线性微分方程

1.形式

​ \(\large\frac{\mathrm{d}^2 y}{\mathrm{d} x^2} + p \cdot \frac{\mathrm{d}y}{\mathrm{d}x} + q·y = 0\)

2.解法

观察方程的形式可以猜测方程的解在求导后形式基本不变

设 \(y = e^{rx}\) 是 \(\large\frac{\mathrm{d}^2 y}{\mathrm{d} x^2} + p \cdot \frac{\mathrm{d}y}{\mathrm{d}x} + q y = 0\) 的解。（\(r\) 是待定常数）

\(\large\frac{\mathrm{d}y}{\mathrm{d}x} = r e^{rx}\)，\(\large\frac{\mathrm{d}^2y}{\mathrm{d}x^2} = r^2 e^{rx}\) 带入原方程中：

\(\large r^2 e^{rx} + p \cdot r \cdot e^{rx} + q \cdot e^{rx} = 0\) \(\longrightarrow\) \(r^2 + pr + q = 0\) \((*)\)

将 \((*)\) 式称为原方程的 特征方程。

特征方程：\(r^2 + pr + q = 0\) 的根 \(\large\frac{\mathrm{d}^2y}{\mathrm{d}x^2} + p \frac{\mathrm{d}y}{\mathrm{d}x} + qy\) 的通解
两个不相等的实根 \(r_1, r_2\) \(y = C_1 e^{r_1x} + C_2 e^{r_2x}\)
两个相等的实根 \(r_1 = r_2 = r\) \(y = (C_1 + C_2 x) e^{rx}\)
两个共轭复根 \(r_1, r_2 = \alpha \pm i\beta\) \(y = e^{\alpha x}(C_1 \cos \beta x + C_2 \sin \beta x)\)

(1) \(p^2 - 4q > 0\) 时

特征方程有两个不相等的实根 \(r_1, r_2\)，则：\(y_1 = e^{r_1x}\)，\(y_2 = e^{r_2x}\)

是原方程的解。又 \(\large\frac{y_1}{y_2} = \frac{e^{r_1x}}{e^{r_2x}} = e^{(r_1-r_2)x} \neq\) 常数，故 \(y_1, y_2\) 线性无关。

由 叠加原理：通解为 \(y = C_1 y_1 + C_2 y_2 = C_1 e^{r_1x} + C_2 e^{r_2x}\)

(2) \(p^2 - 4q = 0\) 时

特征方程有两个相等的实根 \(r_1 = r_2 = r\)。

则：\(y_1 = e^{rx}\) 是原方程的解。（这里还需要再找到另外一个与 \(y_1\) 线性无关的解）

设 \(y = u \cdot e^{rx}\) 是原方程的解，

\(\large\frac{\mathrm{d}y}{\mathrm{d}x} = \frac{\mathrm{d}u}{\mathrm{d}x} e^{rx} + u r e^{rx} = (\frac{\mathrm{d}u}{\mathrm{d}x} + ru) e^{rx}\) \(\large\frac{\mathrm{d}^2y}{\mathrm{d}x^2} = (\frac{\mathrm{d}^2u}{\mathrm{d}x^2} + r \frac{\mathrm{d}u}{\mathrm{d}x}) e^{rx} + (\frac{\mathrm{d}u}{\mathrm{d}x} + ru) r e^{rx}= [\frac{\mathrm{d}^2u}{\mathrm{d}x^2} + (2r) \frac{\mathrm{d}u}{\mathrm{d}x} + r^2u] e^{rx}\)

代入原方程：

\(\large(\frac{\mathrm{d}^2u}{\mathrm{d}x^2} + 2r \frac{\mathrm{d}u}{\mathrm{d}x} + r^2u) e^{rx} + p (\frac{\mathrm{d}u}{\mathrm{d}x} + ru) e^{rx} + q u e^{rx} = 0\large\longrightarrow \frac{\mathrm{d}^2u}{\mathrm{d}x^2} + (2r+p) \frac{\mathrm{d}u}{\mathrm{d}x} + (r^2 + pr + q)u = 0\)

\(\large\frac{\mathrm{d}^2u}{\mathrm{d}x^2} + (2r+p) \frac{\mathrm{d}u}{\mathrm{d}x} + (r^2 + pr + q)u = 0\) \(\longrightarrow\) \(\large\frac{\mathrm{d}^2u}{\mathrm{d}x^2} = 0\) （\(p^2 - 4q = 0\)）

取 \(u = x\)，则 \(y_2 = x e^{rx}\) 是 \((*)\) 的解。

由 \(y_1, y_2\) 线性无关。

由 叠加原理：通解为 \(y = (C_1 + C_2 x) e^{rx}\)

(3) \(p^2 - 4q < 0\) 时

特征方程有一对共轭复根，\(r_1, r_2 = \alpha \pm i\beta\)。

由欧拉公式：\(e^{ix} = \cos x + i \sin x\)　　\(e^{-ix} = \cos(-x) + i \sin(-x) = \cos x - i \sin x\)

\(\large y_1 = e^{r_1x} = e^{(\alpha + i\beta)x} = e^{\alpha x + i\beta x} = e^{\alpha x} (\cos \beta x + i \sin \beta x)\)

\(\large y_2 = e^{r_2x} = e^{(\alpha - i\beta)x} = e^{\alpha x - i\beta x} = e^{\alpha x} (\cos \beta x - i \sin \beta x)\)

令 \(\large y_1^* = \frac{1}{2}(y_1 + y_2) = e^{\alpha x} \cos \beta x\)　　\(y_2^* = \large\frac{1}{2i}(y_1 - y_2) = e^{\alpha x} \sin \beta x\)

\(\large\frac{y_1^*}{y_2^*} = \pm \cot \beta x\)，\(y_1^*, y_2^*\) 线性无关。

由 叠加原理：通解为 \(y = e^{\alpha x}(C_1 \cos \beta x + C_2 \sin \beta x)\)

3.例子

例 1 求 \(\large\frac{\mathrm{d}^2y}{\mathrm{d}x^2} + 3\frac{\mathrm{d}y}{\mathrm{d}x} - 10y = 0\) 通解

解：特征方程为：\(r^2 + 3r - 10 = 0\) \(\longrightarrow\) \((r+5)(r-2) = 0\)

则：\(r_1 = -5\)　　\(r_2 = 2\)

通解为：\(y = C_1 e^{-5x} + C_2 e^{2x}\)，\(C_1, C_2\) 是任意常数

例 2 求 \(\large\frac{\mathrm{d}^2y}{\mathrm{d}x^2} - 4\frac{\mathrm{d}y}{\mathrm{d}x} + 4y = 0\) 通解

解：特征方程为：\(r^2 - 4r + 4 = 0\) \(\longrightarrow\) \((r-2)^2 = 0\)

则：\(r_1 = r_2 = 2\)

通解为：\(y = (C_1 + C_2 x) e^{2x}\)，\(C_1, C_2\) 是任意常数

例 3 求 \(\large\frac{\mathrm{d}^2y}{\mathrm{d}x^2} + 4\frac{\mathrm{d}y}{\mathrm{d}x} + 7y = 0\) 通解

解：特征方程为：\(r^2 + 4r + 7 = 0\)

则：\(r_1 = -2 + \sqrt{3}i\)　　\(r_2 = -2 - \sqrt{3}i\)

通解为：\(y = e^{-2x}(C_1 \cos\sqrt{3}x + C_2 \sin\sqrt{3}x)\)，\(C_1, C_2\) 是任意常数

6.2 二阶常系数非齐次线性微分方程

形式： \(\large \frac{\mathrm{d}^2y}{\mathrm{d}x^2} + p\frac{\mathrm{d}y}{\mathrm{d}x} + qy = f(x)\)

由之前的二阶非齐次线性微分方程的通解的结构定理，我们现在其实只需要关注其特解的解法。

6.2.1 类型一

1.形式

\(f(x) = P_n(x) \cdot e^{\alpha x}\)

其中 \(P_n\) 是一个 \(n\) 次多项式。 即方程为 \(\large \frac{\mathrm{d}^2y}{\mathrm{d}x^2} + p\frac{\mathrm{d}y}{\mathrm{d}x} + qy = P_n(x)e^{\alpha x}\)

2.解法

还是同二阶常系数齐次微分方程的观察法，我们可以猜测其特解的形式。

设 \(\tilde{y} = Q(x)e^{\alpha x}\) 是上述方程的特解 \(Q(x)\) 是一个待定多项式

求导可得： \(\large \frac{\mathrm{d}\tilde{y}}{\mathrm{d}x} = \frac{\mathrm{d}Q}{\mathrm{d}x} \cdot e^{\alpha x} + Q \cdot \alpha \cdot e^{\alpha x} = (\frac{\mathrm{d}Q}{\mathrm{d}x} + \alpha Q)e^{\alpha x}\)

\(\large \frac{\mathrm{d}^2\tilde{y}}{\mathrm{d}x^2} = (\frac{\mathrm{d}^2Q}{\mathrm{d}x^2} + \alpha \frac{\mathrm{d}Q}{\mathrm{d}x})e^{\alpha x} + \alpha(\frac{\mathrm{d}Q}{\mathrm{d}x} + \alpha Q)e^{\alpha x} = (\frac{\mathrm{d}^2Q}{\mathrm{d}x^2} + 2\alpha\frac{\mathrm{d}Q}{\mathrm{d}x} + \alpha^2 Q)e^{\alpha x}\)

带入原方程： \(\large (\frac{\mathrm{d}^2Q}{\mathrm{d}x^2} + 2\alpha\frac{\mathrm{d}Q}{\mathrm{d}x} + \alpha^2 Q)e^{\alpha x} + p(\frac{\mathrm{d}Q}{\mathrm{d}x} + \alpha Q)e^{\alpha x} + q Q e^{\alpha x} = P_n(x)e^{\alpha x}\)

整理得： \(\large \frac{\mathrm{d}^2Q}{\mathrm{d}x^2} + (2\alpha + p)\frac{\mathrm{d}Q}{\mathrm{d}x} + (\alpha^2 + p\alpha + q)Q = P_n(x)\)

根据 \(\alpha\) 与特征方程根的关系（或者也可以理解为上述式子的系数是否为零），分以下三种情况讨论：

(1) \(\alpha^2 + p\alpha + q \neq 0\) ** 时，即 \(\alpha\) 不是** 特征方程的根 这时 \(Q(x)\) 是一个 \(n\) 次多项式，设 \(Q(x) = a_0x^n + a_1x^{n-1} + ... + a_n=Q_n(x)\) 将 \(\tilde{y} = Q_n(x)e^{\alpha x}\) 带入上述关于 \(Q(x)\) 与 \(P_n(x)\) 的方程，可解出 \(Q_n(x)\)
此时特解形式： \(\tilde{y} = Q_n(x)e^{\alpha x}\)

(2) \(\alpha^2 + p\alpha + q = 0\) ** 时，但 \(2\alpha + p \neq 0\)。即 \(\alpha\) 是特征方程的 单根**。 取 \(Q(x) = x Q_n(x)\) 此时特解形式： \(\tilde{y} = x Q_n(x)e^{\alpha x}\)

(3) \(\alpha^2 + p\alpha + q = 0\) ** 时，且 \(2\alpha + p = 0\)。即 \(\alpha\) 是特征方程的 二重根**。 取 \(Q(x) = x^2 Q_n(x)\) 此时特解形式： \(\tilde{y} = x^2 Q_n(x)e^{\alpha x}\)

总结

令 \(k\) 为 \(\alpha\) 是特征方程的根的重数： $$ k = \begin{cases} 0, & \alpha \text{ 不是特征方程的根} \ 1, & \alpha \text{ 是特征方程的单根} \ 2, & \alpha \text{ 是特征方程的二重根} \end{cases} $$

则称 \(\alpha\) 是特征方程的 \(k\) 重根。 方程的特解形式为： $$ \large \tilde{y} = x^k Q_n(x) e^{\alpha x} $$

3.例子

例1 求 \(\large \frac{\mathrm{d}^2y}{\mathrm{d}x^2} + \frac{\mathrm{d}y}{\mathrm{d}x} + 2y = x^2 - 3\) 的一个特解 .

解：特征方程为 \(r^2 + r + 2 = 0\) .

\(r = \frac{-1 \pm \sqrt{1-8}}{2} = -\frac{1}{2} \pm \frac{\sqrt{7}}{2}i\) .

\(\alpha = 0\)，不是特征方程的根 .

则：特解形如 \(\tilde{y} = ax^2 + bx + c\) .

代入原方程： \(2a + (2ax + b) + 2(ax^2 + bx + c) = x^2 - 3\) \(2a + 2ax + b + 2ax^2 + 2bx + 2c = x^2 - 3\)

整理得： $$ 2ax^2 + (2a+2b)x + (2a+b+2c) = x^2 - 3 $$

得： $$ \begin{cases} 2a = 1 &\implies a = \frac{1}{2} \ 2a + 2b = 0 &\implies b = -a = -\frac{1}{2} \ 2a + b + 2c = -3 &\implies c = -\frac{7}{4} \end{cases} $$

故特解 \(\tilde{y} = \frac{1}{2}x^2 - \frac{1}{2}x - \frac{7}{4}\)

例2 求 \(\large \frac{\mathrm{d}^2y}{\mathrm{d}x^2} + y = (x-2)e^{3x}\) 的通解 .

解：特征方程为 \(r^2 + 1 = 0 \implies r = \pm i\) .

所以对应的齐次方程通解为：\(Y = C_1\cos x + C_2\sin x\) .

又：\(\alpha = 3\) 不是特征方程的根 .

设原方程特解为 \(\tilde{y} = (ax+b)e^{3x}\) .

代回原方程中（或者前文中 \(Q(x)\) 与 \(P(x)\) 的方程）：

\(10(ax+b) + 6a = x-2\) .

$ \begin{cases} 10a = 1 \implies a = \frac{1}{10} \ 10b + 6a = -2 \implies 10b = -2 - \frac{6}{10} = -\frac{26}{10} \implies b = -\frac{13}{50} \end{cases} $

\(\longrightarrow \tilde{y} = (\frac{1}{10}x - \frac{13}{50})e^{3x}\)

原方程通解为 \(y = Y + \tilde{y} = C_1\cos x + C_2\sin x + (\frac{1}{10}x - \frac{13}{50})e^{3x}\)

其中 \(C_1, C_2\) 为任意常数.

6.2.1 类型二

\[ f(x) = P_n(x) e^{\alpha x} \cos \beta x \quad (\text{或 } P_n(x) e^{\alpha x} \sin \beta x) \]

即方程为 $\large \frac{\mathrm{d}^2y}{\mathrm{d}x^2} + p\frac{\mathrm{d}y}{\mathrm{d}x} + qy = f(x) = P_n(x) e^{\alpha x} \cos \beta x \quad (\text{或 } P_n(x) e^{\alpha x} \sin \beta x) $

1.解法一：复数法
\[ \begin{cases} \frac{\mathrm{d}^2y}{\mathrm{d}x^2} + p\frac{\mathrm{d}y}{\mathrm{d}x} + qy = P_n(x) e^{\alpha x} \cos \beta x \quad (1) \\ \frac{\mathrm{d}^2y}{\mathrm{d}x^2} + p\frac{\mathrm{d}y}{\mathrm{d}x} + qy = P_n(x) e^{\alpha x} \sin \beta x \quad (2) \end{cases} \]

考虑辅助方程： $$ \frac{\mathrm{d}^2y}{\mathrm{d}x^2} + p\frac{\mathrm{d}y}{\mathrm{d}x} + qy = P_n(x) e^{\alpha x + i\beta x} = P_n(x)e^{\alpha x}(\cos \beta x + i\sin \beta x) \quad (3) $$

设 \((3)\) 有特解 \(\tilde{y} = \tilde{y}_1 + i\tilde{y}_2\)，则 \(\tilde{y}_1\) 是方程 \((1)\) 的特解，\(\tilde{y}_2\) 是方程 \((2)\) 的特解 .

2.例题

**例1 ** 求 \(\large \frac{\mathrm{d}^2y}{\mathrm{d}x^2} - y = \sin x\) 通解 .

解：特征方程 \(r^2 - 1 = 0 \longrightarrow r = \pm 1\) .

\(\therefore\) 对应齐次方程解为 \(Y = C_1 e^x + C_2 e^{-x}\) .

构造复数方程 \(\frac{\mathrm{d}^2y}{\mathrm{d}x^2} - y = e^{ix} = \cos x + i\sin x\) .

设其特解 \(\tilde{y}^* = a e^{ix}\)，带回原方程中： \(a(i)^2 e^{ix} - a e^{ix} = e^{ix}\) \(-a - a = 1 \longrightarrow a = -\frac{1}{2}\)

\(\therefore \tilde{y}^* = -\frac{1}{2} e^{ix} = -\frac{1}{2}(\cos x + i\sin x)\) .

原方程右端为 \(\sin x\)（即 \(e^{ix}\) 的虚部），故取特解为虚部： \(\tilde{y} = \text{Im}(\tilde{y}^*) = -\frac{1}{2} \sin x\).

故通解为 \(y = Y + \tilde{y} = C_1 e^x + C_2 e^{-x} - \frac{1}{2}\sin x\)，\(C_1, C_2\) 为任意常数.

例2 求 \(\large \frac{\mathrm{d}^2y}{\mathrm{d}x^2} - y = e^x \cos 2x\) 通解 .

解：特征方程为 \(r^2 - 1 = 0, \quad r = \pm 1\) .

对应齐次方程解为 \(Y = C_1 e^x + C_2 e^{-x}\) .

构造方程 \(\frac{\mathrm{d}^2y}{\mathrm{d}x^2} - y = e^{x+i2x} = e^{(1+2i)x} = e^x(\cos 2x + i\sin 2x)\) .

设其特解为 \(\tilde{y}^* = a e^{(1+2i)x}\)，带回原方程中：

\[ \frac{\mathrm{d}^2}{\mathrm{d}x^2}(a e^{(1+2i)x}) = a(1+2i)^2 e^{(1+2i)x} = a(1+4i-4)e^{(1+2i)x} = a(-3+4i)e^{(1+2i)x} \]

代入得： \(a(-3+4i) - a = 1\) \(a(-4+4i) = 1 \longrightarrow a = \frac{1}{-4+4i} = -\frac{1}{8}(1+i)\)

此时复数特解 \(\tilde{y}^* = -\frac{1}{8}(1+i)e^{(1+2i)x}\). 原方程右端为 \(e^x \cos 2x\)（实部），故取实部解. $$ \begin{aligned} \tilde{y}^* &= \left[ -\frac{1}{8}(1+i)e^{(1+2i)x} \right] \ &= \left[ -\frac{1}{8}(1+i)e^x (\cos 2x + i\sin 2x) \right] \ &= -\frac{1}{8}e^x \left[ (\cos 2x + i\sin 2x + i\cos 2x - \sin 2x) \right] \ \end{aligned} $$

故方程特解 \(\tilde{y} = -\frac{1}{8}e^x(\cos 2x - \sin 2x)\) .

故通解为 \(y = Y + \tilde{y} = C_1 e^x + C_2 e^{-x} - \frac{1}{8}e^x(\cos 2x - \sin 2x)\) ,\(C_1, C_2\) 为任意常数.

3.解法二：
\[ \large \frac{\mathrm{d}^2y}{\mathrm{d}x^2} + p \frac{\mathrm{d}y}{\mathrm{d}x} + q = P_m(x) e^{\alpha x} \cos\beta x + Q_n(x) e^{\alpha x} \sin\beta x \]

设 \(\alpha + \beta i\) 是特征方程的 \(k\) 重根，则特解形式如下：

$$ \large \tilde{y} = x^k [R_h(x)e^{\alpha x}\cos\beta x + S_h(x)e^{\alpha x}\sin\beta x] $$ $h=max\left { m,n \right } $

4.例题

例3 求 \(\large \frac{\mathrm{d}^2y}{\mathrm{d}x^2} - y = e^x \cos 2x\) 的通解.

解：\(r^2 - 1 = 0, \quad r = \pm 1\).

对应的特征方程通解为 \(Y = C_1 e^x + C_2 e^{-x}\).

\(1+2i\) 不是特征方程的根. \(k=0\).

设 \(\tilde{y} = a e^x \cos 2x + b e^x \sin 2x\) 为特解.

\(\frac{\mathrm{d}\tilde{y}}{\mathrm{d}x} = [(a+2b)\cos 2x + (-2a+b)\sin 2x] e^x\).

\(\frac{\mathrm{d}^2\tilde{y}}{\mathrm{d}x^2} = [(-4a-3b)\sin 2x + (-3a+4b)\cos 2x] e^x\).

带入原方程：

\((-4a-3b)\sin 2x + (-3a+4b)\cos 2x - a\cos 2x - b\sin 2x = \cos 2x\).

\(\longrightarrow \begin{cases} -4a-4b=0 \\ -4a+4b+1=0 \end{cases} \longrightarrow \begin{cases} a=-\frac{1}{8} \\ b=\frac{1}{8} \end{cases}\).

\(\tilde{y} = -\frac{1}{8}e^x \sin 2x + \frac{1}{8}e^x \cos 2x\).

通解为 \(y = Y + \tilde{y} = C_1 e^x + C_2 e^{-x} - \frac{1}{8}e^x \cos 2x + \frac{1}{8}e^x \cos 2x\).

例4 求 \(\large \frac{\mathrm{d}^2y}{\mathrm{d}x^2} - y = e^x \cos 2x + \sin x\) 的通解.

解：

(1)\(\frac{\mathrm{d}^2y}{\mathrm{d}x^2} - y = 0\) 的通解为 \(Y = C_1 e^x + C_2 e^{-x}\).

(2) \(\frac{\mathrm{d}^2y}{\mathrm{d}x^2} - y = e^x \cos 2x\) 的特解为 \(\tilde{y}_1\).

(3) \(\frac{\mathrm{d}^2y}{\mathrm{d}x^2} - y = \sin x\) 的特解为 \(\tilde{y}_2\).

所求通解为 \(y = Y + \tilde{y}_1 + \tilde{y}_2\).

6.3 高阶常系数非齐次线性微分方程的一些例子

1.例子

例 1 \(y''' + 3y'' + 3y' + y = e^x\).

解：特征方程为：\(r^3 + 3r^2 + 3r + 1 = 0 \longrightarrow (r+1)^3 = 0, \quad r = -1\).

对应的齐次方程通解为 \(Y = (C_1 + C_2x + C_3x)e^{-x}\).

\(\alpha = 1\) 不是特征方程的根.

设特解为 \(\tilde{y} = \alpha e^x\).

代入原方程中：

\(\alpha + 3\alpha + 3\alpha + \alpha = 1 \longrightarrow \alpha = \frac{1}{8}\).

\(\tilde{y} = \frac{1}{8}e^x\).

原方程通解为 \(y = Y + \tilde{y} = (C_1 + C_2x + C_3x)e^{-x} + \frac{1}{8}e^x\).
